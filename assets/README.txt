LOGO SETUP INSTRUCTIONS
-----------------------
1. Save the Nusa Meals logo image as:
   assets/logo.png

   (The file must be named exactly "logo.png" inside this "assets" folder)

2. The logo is referenced in index.html at:
   <img src="assets/logo.png" alt="Nusa Meals" />

3. Supported formats: PNG (recommended), JPG, or WebP
   Recommended logo height: 44px display height (any resolution works)

Once the file is in place, open index.html in your browser and the logo
will appear in the navbar and footer automatically.
